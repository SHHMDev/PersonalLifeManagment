# Personal Life Management App

## Purpose

This application is a personal life management tool for a single user.

The goal is not team collaboration.

The goal is helping the user manage:

* Goals
* Projects
* Tasks
* Recurring Duties
* Daily Logs

The application language is Persian (Farsi).

All visible texts inside UI must be written in Persian.

Variable names, source code, file names and architecture must remain in English.

---

## Platform

* Android
* TypeScript
* Capacitor

APK output is required.

* Rounded icons

* Soft shadow

---

## User Experience

The application is for daily use.

Fast access is more important than advanced features.

Avoid unnecessary complexity.

---

## Categories

Categories are NOT predefined.

User creates categories dynamically.

Example:

Goals:

* Spiritual
* Business
* Education

Tasks:

* Family
* Work
* Education

Projects:

* Programming
* Business

Every module manages its own categories.

No hardcoded categories.

---

UI language must be Persian.

All labels, buttons and messages must be Persian.
