# مدیریت زندگی شخصی - Android (Capacitor + TypeScript)

## شروع سریع

```bash
npm install
npm run build
npm run cap:sync
npm run cap:open:android
```

## نکات کلیدی

- رابط کاربری فارسی و RTL
- طراحی مینیمال با رنگ آبی/سفید
- معماری ماژولار برای Goals/Projects/Tasks/Recurring/Daily Logs
- آماده برای SQLite و ساخت APK
